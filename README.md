# Full-Stack Job Portal Application

This is a full-stack job portal application built with Node.js/Express for the backend and React for the frontend.

## Features
- User Authentication (Login/Signup)
- Job Listings
- Job Details
- Dashboard

## How to Run locally

### Backend
1. Go to `Backend` directory
2. Run `npm install`
3. Run `npm start`

### Frontend
1. Go to `Frontend/jobportalapplication` directory
2. Run `npm install`
3. Run `npm start`
